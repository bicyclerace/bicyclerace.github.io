Data

The data used for this project comes from a "cleaned up" version of the original Divvy data made available by Steven Yance.

The data includes information about all the Divvy trips that were taken between 26 Jun 2013 to 31 Dec 2013.

Importantly, Divvy data collection began when the Divvy service was just started and plenty of stations weren't open yet.

As downloaded, the data was quite clean, and only minor pre-processing was necessary.

Hosting

The data are stored on a MySQL database hosted on UIC's personal web hosting service. To access the database, a set of PHP API calls were developed.

Weather data

Historical weather data about Chicago was retrieved from Weather Underground. Weather Underground returns a huge amount of weather data for the period of the 1st of July to December 31, and since most of that data was not useful for the project purposes it was filtered. In particular, our project used weather observations at particular intervals throughout the day, with information about the time of the observation, the conditions ("Clear", "Cloudy", ..) and the temperature in both fahrenheit and celsius.

We also kept the information about daily summaries about min and max temperature, and boolean values that indicates whether certain conditions (fog, rain, ..) have occurred during the day. The latter information, though, has not been yet been used for the project.

Sunrise and sunset

Sunrise and sunset are computed using SunCalc, a tiny JavaScript library for calculating sun/moon positions and phases.
