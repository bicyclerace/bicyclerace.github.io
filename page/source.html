Source code

Source code of the web application can be find at https://github.com/bicyclerace/bicyclerace.github.io.

Install instructions

To run the web app on localhost, you need to have a running local server environment active. You can download MAMP, install it and start the server. Then you need to put your source code under MAMP "htdoc" folder.

Credits

For the project we have used D3 for manipulating documents based on data, Leaflet.js for interactive maps, jQuery for DOM manipulation, Underscore.js which provides useful functional programming helpers, and SunCalc a library for calculating sun/moon positions and phases.
